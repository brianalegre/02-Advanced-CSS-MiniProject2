Flexbox Row

Use these three properties to create a Flexbox row layout.

    .row {
        display: flex;
        flex-direction: row;
        flex-wrap: wrap;
    }

Flexbox Column

Use this to create a Flexbox column layout.

    .column {
        display: flex;
        flex-direction: colmn;
    }
